# GlassGPT 5.4.0 Release Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restore the 4.12.6 user experience for image upload, file upload, tool call indicators, and sandbox file downloads on the backend-owned 5.4.0 architecture, then publish backend production and iOS TestFlight build 20223 from `codex/stable-5.4`.

**Architecture:** Keep the 250 ms polling architecture and backend-owned OpenAI key. Fix attachment regressions by correcting route body limits and attachment-only message encoding, fix tool indicators by retaining grace-period live state across run finalization, and wire generated-file download/preview through the existing cache and preview primitives that are currently disconnected from the chat surfaces.

**Tech Stack:** Swift 6 / iOS 26, Cloudflare Workers (Hono), OpenAI Responses API, D1, Durable Objects, SwiftData, Vitest, Xcodebuild
